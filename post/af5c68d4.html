Pandas--DataFrame

发表于2022-05-20|更新于2022-12-04|Python
|阅读量:

DataFrame是一个表格型结构，既有行标签，又有列标签，他也被称为异构数据，所谓异构，指的是表格中每列的数据类型不同,比如：有一列是字符串，也可能有一列是整型或者浮点型

上述表格中每列的数据类型：

name age gender rating
String integer String Float

创建DataFrame对象

参数说明：
data：输入的数据，可以是一个ndarray，series，list，dict，也可以是一个DataFrame
index：行标签，如果不传值，默认从0开始
columns：行标签：如果不传值，默认从0开始
dtype：每一列的数据类型
copy：默认为False，表示复制数据data

三、index查增删

1.查：loc[“标签索引”]函数

注：该函数允许传递两个参数行和列，之间用逗号隔开，而且只能使用标签索引：df.loc[“a”,”one”]

print(df.loc["a"])

2. 查：iloc[“整数索引”]函数

注：该函数允许传递两个参数行和列，之间用逗号隔开，而且只能使用整数索引：df.iloc[“2”,”0”]

print(df.iloc[2])

3.查：切片

# 左闭右开
print(df[2:4]) # 意思：第三行

4. 添加：append（）函数

使用 append() 函数，可以将新的数据行添加到 DataFrame 中，该函数会在行末追加数据行。

5. 删除：drop（）函数

您可以使用行索引标签，从 DataFrame 中删除某一行数据。如果索引标签存在重复，那么它们将被一起删除
