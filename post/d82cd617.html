insert into或insert ignore into或replace into

发表于2022-11-02|更新于2023-07-20|MySQL
|阅读量:

总结：

指令 已存在（判断条件为unique） 不存在 举例
insert 报错 插入 insert into User (name,age) values(‘hwang1’,10);
insert ignore 忽略 插入 insert ignore into User (name,age) values(‘hwang1’,10);
replace 替换 插入 replace into User (name,age) values(‘hwang1’,10);

insert into：

当上述数据存在时执行以下sql，id会自增，但是插入不成功，会报错；

insert ignore into

当上述数据存在时，且多加一个ignore时，忽略插入的记录，id会自增，不会报错，返回一个0

replace into

当上述数据存在时，则会先删除已存在的数据，再添加新的数据
