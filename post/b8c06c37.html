使用Field定制字段

发表于2022-11-23|更新于2022-12-06|Python
|阅读量:

简介

Field函数提供关于字段和验证的额外信息，如设置必填项和可选，设置最大值和最小值，字符串长度等限制

参数

这里列举了一些常用的参数：

  • Field(default)：默认值，可为None，可用省略号（…）来表示字段必填
  • alias：字段的公开名称。
  • title：自定义标题，如果没有默认就是字段属性的值
  • description：定义字段描述内容
  • gt：对于数值值 (int，float，Decimal)，将向 JSON 模式添加一个 “大于” 验证和一个 exclusiveMinimum。
  • ge：对于数值值 ，将向 JSON 模式添加一个 “大于等于” 验证和一个 minimum。
  • lt：对于数值值 ，将向 JSON 模式添加一个”小于” 验证和一个 exclusiveMaximum 。
  • le：对于数值值 ，将向 JSON 模式添加一个 “小于等于” 验证和一个 maximum 。
  • min_length：对于字符串值，将向 JSON 模式添加相应的验证和 minLength。
  • max_length：对于字符串值，将向 JSON 模式添加相应的验证和 maxLength。
  • exclude：在转储（.dict和.json）实例时排除此字段
  • include：在转储（.dict和.json）实例时（仅）包含此字段
  • 等。。。参考：https://cloud.tencent.com/developer/article/1949684
