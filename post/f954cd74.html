click命令行神器

发表于2022-12-01|更新于2022-12-04|Python
|阅读量:

介绍

click可以将函数装饰为命令行接口，支持高度自定义的配置。

使用Click 分为两个步骤：

  1. 使用 @click.command() 装饰一个函数，使之成为命令行接口
  2. 使用 @click.option() 装饰函数，为其添加可选命令行参数，支持设置固定长度的参数值
  3. 使用 @click.argument() 装饰函数，为其添加固定命令行参数，支持设置不定长度的参数值

@click.option() 最基本的用法就是通过指定命令行选项的名称，从命令行读取参数值，再将其传递给函数。常用参数值如下：

  • default：设置命令行参数的默认值
  • help：参数说明
  • type：参数类型，可以是 string, int, float 等
  • choice：可选值，可选多个值
  • echo：输出，和print一样
  • nargs：指定命令行参数接收的值的个数，他就是option()支持设置固定长度的参数值
  • hide_input：用于隐藏输入
  • confirmation_promt：用于重复输入
  • prompt：当在命令行中没有输入相应的参数时，会根据 prompt 提示用户输入
  • is_eager：True表明该命令行选项优先级高于其他选项
  • expose_value：False表示如果没有输入该命令行选项，会执行既定的命令行流程
  • callback：回调函数，指定了输入该命令行选项时，要跳转执行的函数
  • metavar：如何在帮助页面表示值

@click.argument()常见的参数值如下：

  • nargs：布丁参数，nargs=-1时，表明参数接收不定量的参数值，参数值会以 tuple 的形式传入函数；如果 nargs 大于等于 1，表示接收 nargs 个参数值
