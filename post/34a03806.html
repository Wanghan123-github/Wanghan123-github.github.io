python2和python3的区别

发表于2023-07-27|更新于2023-08-23|Python
|阅读量:

01、代码特点

  • Python 2于2000年10月16日发布，稳定版本是Python 2.7。Python 3于2008年12月3日发布，不完全兼容Python 2。
  • python2 的代码混乱，重复较多，冗余。
  • python3源码规范、清晰、简单优美。

02、代码输出差异

  • python2：print ‘内容’

  • python3：print（”内容”）

03、代码输入差异

  • python2：raw-input()
  • python3：input ()

04、range、xrange的变化

  • Python2：使用xrange()来创建一个迭代器对象，使用range()创建一个list数组；
  • Python3：使用range()创建迭代器对象，移除了xrange()方法。

05、字典

  • Python2：dict.items()、dict.keys() 和 dict.values() 返回的是列表

  • Python3：返回的是对象

06、捕获异常

  • Python2：捕获异常的语法是 except Exception, e:
  • Python3：捕获异常的语法是 except Exception as e:

07、函数返回

  • Python2： range()、map()、filter()返回列表

  • Python3： range()、map()、filter()返回迭代器

08、编码的改变

  • Python2：字符串被分为两种类型，

    • 一种ASCII字符串，使用ASCII编码，每个字符使用一个字节表示。
    • 一种Unicode字符串，使用 Unicode编码（UTF-16），以u开头，每个字符使用 2 个字节表示。默认ASCII编码方式。

  • Python3：所有字符串都是 Unicode 字符串。默认的编码方式是UTF-8，一种可变长度的Unicode 编码方式。

09、unicode不同

  • Python2：Unicode 默认2个字节表示一个字符

  • Python3：Unicode 字符的字节表示数量取决于具体字符在 Unicode 中的编码范围。最小的情况是一个字节，最多可能是四个字节（针对某些特殊字符）

10、nonlocal关键字

  • Python2：没有nonlocal关键字
  • Python3：加入nonlocal（允许在一个函数中修改其外部函数的局部变量，也就是在嵌套函数之内使用嵌套函数外的局部变量）

11、__init__文件

  • Python2：新建的包如果没有init文件 则包不能够被调用，会直接报错。

  • Python3：新建的包里面的init文件，不再是强制性的。如果你删除该文件 包照样可以被调用。但是通常建议保留，因为里面可能包含初始化的代码
