Redis基础命令

发表于2022-11-07|更新于2023-08-14|Redis
|阅读量:

Redis服务

  • brew services start redis：启动服务

  • redis-server ：启动服务

  • ps axu | grep redis ：查看服务进程

  • redis-cli：连接redis服务

  • brew services stop redis：关闭服务

  • redis-cli shutdown：关闭服务

  • kill -9 ***：关闭服务

字符串操作命令

  • set key value：设置键的值。
  • get key：获取键的值。
  • del key：删除指定的键。
  • incr key：将键存储的整数值增加 1。
  • incrby key increment：将键存储的值分别增加指定的数量（increment）
  • decr key：将键存储的整数值减少 1。
  • decrby key decrement：将键存储的值分别减少指定的数量（decrement）
  • append key value：将值追加到键的值后面。
  • strlen key：获取键存储的值的长度。
  • getrange key start end：获取键对应的值，并截取start到end的长度
  • setrange key offset value：将键对应存储的值中offset的位置，替换为value
  • mget key1 key2...：获取多个键的值
  • mset key1 value1 key2 value2 ...]：设置多个键的值
  • msetnx key1 value1 key2 value2 ...：设置多个键的值，仅在所有键都不存在时才设置。
  • getset key value：设置键的新值并返回旧值。
  • setnx key value：仅在键不存在时设置键的值。
  • strcmp key1 key2：比较两个键存储的值。
  • setbit key offset value：设置键的指定位偏移位置的值。
  • getbit key offset：获取键的指定位偏移位置的值。

哈希

  • hset key field value：在哈希数据类型中设置字段的值。
  • hget key field：获取哈希数据类型中指定字段的值。
  • hdel key field1 field2 ... ：删除一个或多个哈希数据类型中指定字段。
  • hgetall key：获取哈希中所有字段和值的列表。
  • hkeys key：获取哈希中所有字段的列表。
  • hvals key：获取哈希中所有值的列表。
  • hincrby key field increment：将哈希中指定字段的值增加指定的增量。
  • hexists key field：检查哈希中是否存在指定字段。
  • hlen key：获取哈希中字段的数量。
  • hmget key field1 field2 ...：获取哈希中多个字段的值。
  • hmset key field1 value1 field2 value2 ...：设置哈希中多个字段的值。
  • hsetnx key field value：仅在哈希中指定字段不存在时设置值。
  • hstrlen key field：获取哈希中指定字段值的长度。

列表

  • lpush key value1 value2 ...：将一个或多个值插入到列表的头部。
  • rpush key value1 value2 ...：将一个或多个值插入到列表的尾部。
  • lpush key：移除并返回列表的第一个元素。
  • lpop key：移除并返回列表的头部元素。
  • rpop key：移除并返回列表的最后一个元素。
  • llen key：获取列表的长度（即包含的元素个数）。
  • lrange key start stop：获取列表中指定范围的元素。
  • lindex key index：获取列表中指定索引位置的元素。
  • lset key index value：将列表中指定索引位置的元素设置为新值。
  • lrem key count value：从列表中删除指定数量的指定值。
  • ltrim key start stop：截取列表，只保留指定范围的元素。
  • rpoplpush source destination：将一个元素从源列表尾部移除并添加到目标列表头部。
  • blpop key1 [key2 ...] timeout：阻塞式弹出，等待并返回列表中的元素。（从左边的列表头部弹出元素，如果列表为空，则阻塞等待，直到超时为止）
  • brpop key1 [key2 ...] timeout：从右边的列表尾部弹出元素，如果列表为空，则阻塞等待，直到超时为止。
  • linsert key before pivot value：在列表中某个元素前插入一个值。
  • linsert key after pivot value：在列表中某个元素后插入一个值。

集合

  • sadd key member1 member2 ...：将一个或多个成员添加到集合中。
  • srem key member1 member2 ...：从集合中移除一个或多个成员。
  • smembers key：获取集合中的所有成员。
  • **sismember key member**：检查成员是否存在于集合中。
  • **scard key**：获取集合中的成员数量。
  • **spop key [count]**：随机移除并返回集合中的一个或多个成员。
  • **srandmember key [count]**：随机获取集合中的一个或多个成员。
  • **sdiff key [key ...]**：计算多个集合的差集。
  • **sinter key [key ...]**：计算多个集合的交集。
  • **sunion key [key ...]**：计算多个集合的并集。
  • **sdiffstore destination key [key ...]**：将多个集合的差集存储到新的集合。
  • **sinterstore destination key [key ...]**：将多个集合的交集存储到新的集合。
  • **sunionstore destination key [key ...]**：将多个集合的并集存储到新的集合。

有序集合

  • **zadd key [nx|xx] [ch] [incr] score member [score member ...]**：将一个或多个成员及其分数添加到有序集合中。
    • nx：仅在成员不存在时才添加。
    • xx：仅在成员已经存在时才更新。
    • ch：返回修改的成员数量，不包括新增的成员。
    • incr：对于每个成员，都会将其现有的分数与新的分数相加。
  • **zrem key member [member ...]**：从有序集合中移除一个或多个成员。
  • **zcard key**：获取有序集合中的成员数量。
  • **zscore key member**：获取有序集合中指定成员的分数。
  • **zrank key member**：获取有序集合中指定成员的排名（从小到大排序）。
  • **zrevrank key member**：获取有序集合中指定成员的排名（从大到小排序）。
  • **zrange key start stop [withscores]**：获取有序集合中指定排名范围的成员。
    • withscores：同时返回成员的分数。
  • **zrevrange key start stop [withscores]**：获取有序集合中指定排名范围的成员（倒序）。
  • **zincrby key increment member**：将有序集合中指定成员的分数增加指定的增量。
  • **zcount key min max**：获取有序集合中分数在指定范围内的成员数量。
  • **zrangebyscore key min max [withscores] [limit offset count]**：获取有序集合中分数在指定范围内的成员。
  • **zrevrangebyscore key max min [withscores] [limit offset count]**：获取有序集合中分数在指定范围内的成员（倒序）。
  • **zremrangebyrank key start stop**：移除有序集合中指定排名范围内的成员。
  • **zremrangebyscore key min max**：移除有序集合中分数在指定范围内的成员。

键操作命令

  • exists key：检查键是否存在。

  • del key：删除指定的键。

  • ttl key：获取键的生存时间（剩余时间）。

  • type key：获取键存储的值的数据类型。

  • keys w*：获取以“w”开头的键名

  • flushall：清空所有数据库中的数据。
