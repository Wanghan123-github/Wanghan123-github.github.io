内置模块itertools

发表于2023-03-23|更新于2023-03-24|Python
|阅读量:

简介

itertools 是 Python 标准库中提供的一个工具模块，包含了很多用于迭代器和生成器操作的函数。它可以用于快速高效地构建迭代器和生成器，提高程序的效率和可读性。

其中itertools主要来分为三类函数，分别为无限迭代器、输入序列迭代器、组合生成器

无限迭代器

1. count(start=0, step=1)：生成一个从 start 开始、以 step 为步长的无限迭代器。

如果省略了start则默认从0开始，步长默认为1。

2. cycle(iterable)：无限地重复 iterable 中的元素

3. repeat(elem, n=None)：将一个元素elem重复 n 次，如果n 为 None 则无限重复。

组合迭代器

1. product(*iterables, repeat=1)：求多个迭代器的笛卡尔积，repeat 表示重复次数。

repeat参数：

2. permutations(iterable, r=None)：返回的是iterable中长度为r的排列。r默认长度为None即iterable的长度，并且是按顺序返回的，且不包含重复的结果，除非iterable内有重复的元素。

3. combinations(iterable, r)：返回的是iterable中长度为r的组合。与上面的permutations不同：所谓排列，就是指从给定个数的元素中取出指定个数的元素进行排序。组合则是指从给定个数的元素中仅仅取出指定个数的元素，不考虑排序。

4. combinations_with_replacement(iterable, r)：求 iterable 中长度为 r 的组合，允许重复元素。
