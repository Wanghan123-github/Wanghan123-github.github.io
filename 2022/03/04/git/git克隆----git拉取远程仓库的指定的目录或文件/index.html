git克隆----git拉取远程仓库的指定的目录或文件

发表于2022-03-04|更新于2022-11-27|git
|阅读量:

  1. 创建xx文件夹

  2. git init

  3. git config core.sparseCheckout true #打开配置

  4. echo ‘drawPic/MP3000A/*’ >> .git/info/sparse-checkout

    # drawPic/MP3000A/* 为分支下的绝对路径,.git/info/sparse-checkout 不变

  5. git remote add origin xxxxx.git # 设置远程仓库

  6. git pull origin 分支

此时就会在你创建的目录下把你想拉取的远程目录拉下来
