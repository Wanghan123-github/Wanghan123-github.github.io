git忽略----.gitignore忽略文件或文件夹

发表于2022-03-03|更新于2022-11-18|git
|阅读量:

当git提交后，发现没有过滤一些非必要上传的文件，比如.git/.idea/pycache/这些文件夹

我们需要在项目的根目录下创建.gitignore文件

.gitignore

文件写好之后，先不着急提交，这里有一个小技巧

需要先执行一下命令，从仓库先把这些要忽略上传的文件删除

最后：

git add .

git commit -m “add .gitignore”

git push origin 你的分支
